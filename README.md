# Dyson Cube Project

A Minecraft mod for NeoForge 21.1+ that allows you to harvest the power of the sun by building a Dyson Sphere!

## Features

- Build massive Dyson Spheres around the sun
- EM Rail Ejector for launching components
- Ray Receiver for collecting power
- Configurable power generation and storage

## Requirements

- Minecraft 1.21.1
- NeoForge 21.1.208 or later
- Java 21

## Building

```bash
./gradlew build
```
